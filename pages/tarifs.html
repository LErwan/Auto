Permis B

L'auto-école vous propose 3 forfaits pour le Permis B: forfait basique de 20h ou forfait étendu de 25h ou 30h. Vous bénéficiez de 3 mois de code en ligne ainsi que 1 année de cours de code à l'Auto école (Voir les horaires). Pour l'apprentissage un livre de code et un livret d'apprentissage vous sont offerts. Pour plus de détails veuillez consulter les différentes cartes proposées selon le forfait désiré.
